# MN News - Система управления новостями

## Описание
Веб-приложение для публикации и управления новостями с админ-панелью.

## Запуск сервера

1. Откройте терминал в папке `server`:
```bash
cd "C:\Users\PC\Documents\MNnews-main (2)\MNnews-main\server"
```

2. Установите зависимости (если еще не установлены):
```bash
npm install
```

3. Запустите сервер:
```bash
npm start
```

Сервер запустится на `http://localhost:3000`

## Функции админ-панели

### Вход как администратор
1. Перейдите на страницу входа
2. Войдите с учетными данными администратора
3. Убедитесь, что в localStorage установлен флаг `is_admin = 1`

### Управление новостями

#### Создание новости
1. Перейдите в админ-панель: `/pages/menu_pages/admin_panel.html`
2. Нажмите кнопку "+ Создать новую новость"
3. Заполните поля:
   - Заголовок
   - Ссылка на изображение (URL)
   - Текст новости (абзацы разделяются пустой строкой)
4. Нажмите "Сохранить изменения"

#### Редактирование новости
**Способ 1:** Через админ-панель
1. В админ-панели нажмите "Изменить" рядом с нужной новостью
2. Внесите изменения
3. Нажмите "Сохранить изменения"

**Способ 2:** Со страницы новости
1. Откройте новость для просмотра
2. Если вы администратор, внизу появятся кнопки редактирования
3. Нажмите "✏️ Редактировать новость"
4. Внесите изменения и сохраните

#### Удаление новости
**Способ 1:** Через админ-панель
1. В админ-панели нажмите "Удалить" рядом с новостью
2. Подтвердите удаление

**Способ 2:** Со страницы новости
1. Откройте новость
2. Нажмите "🗑️ Удалить новость"
3. Подтвердите удаление

### Обновление списка новостей

#### Автоматическое обновление
После создания, редактирования или удаления новости список автоматически обновляется в админ-панели.

#### Ручное обновление на главной странице
На главной странице (`index.html`) есть кнопка "🔄 Обновить список новостей" для принудительного обновления.

## Технические детали

### Структура данных новости
```json
{
  "id": 1,
  "title": "Заголовок",
  "content": ["Абзац 1", "Абзац 2", "Абзац 3"],
  "image": "https://example.com/image.jpg",
  "date": "2025-05-29T00:00:00.000Z",
  "category": "current",
  "preview": "Краткое описание...",
  "lead": "Вводный текст новости"
}
```

### API Endpoints

#### Новости
- `GET /api/news` - Получить все новости
- `GET /api/news?category=current` - Получить новости категории
- `GET /api/news/:id` - Получить конкретную новость
- `POST /api/news` - Создать новость (требует авторизации админа)
- `PUT /api/news/:id` - Обновить новость (требует авторизации админа)
- `DELETE /api/news/:id?actor=username` - Удалить новость (требует авторизации админа)

#### Аутентификация
- `POST /api/auth/login` - Вход
- `POST /api/auth/register` - Регистрация
- `GET /api/auth/profile/:username` - Получить профиль

## Устранение неполадок

### Новости не загружаются
1. Убедитесь, что сервер запущен (`npm start`)
2. Проверьте консоль браузера на наличие ошибок
3. Проверьте, доступен ли API: `http://localhost:3000/api/health`

### Не работает редактирование
1. Убедитесь, что вы вошли как администратор
2. Проверьте, что `localStorage.getItem('is_admin') === '1'`
3. Проверьте консоль на ошибки запросов

### Изменения не отображаются
1. Нажмите кнопку "Обновить список новостей" на главной странице
2. Обновите страницу браузера (F5)
3. Очистите кэш браузера (Ctrl+Shift+R)
