# 🚀 Быстрая справка по управлению новостями

## ⚡ Быстрый старт

```bash
# Запуск сервера
cd server
npm start

# Открыть приложение
# http://localhost:3000
```

## 🔐 Вход как администратор

**Способ 1 (через консоль):**
```javascript
// F12 → Console → выполнить:
localStorage.setItem('name', 'admin');
localStorage.setItem('is_admin', '1');
// Обновить страницу
```

**Способ 2 (через форму):**
- Перейти на страницу входа
- Ввести данные админа
- Нажать "Войти"

## 📝 Создание новости

1. Админ-панель → "+ Создать новую новость"
2. Заполнить:
   - **Заголовок**: краткое название
   - **Изображение**: URL картинки
   - **Текст**: содержимое (абзацы через пустую строку)
3. Сохранить

**Пример текста:**
```
Первый абзац новости.

Второй абзац с деталями.

Третий заключительный абзац.
```

## ✏️ Редактирование новости

**Способ 1:** Админ-панель → кнопка "Изменить"

**Способ 2:** Страница новости → кнопка "✏️ Редактировать"

**Способ 3:** Прямая ссылка:
```
/pages/menu_pages/admin_panel.html?edit=ID
```

## 🗑️ Удаление новости

**Способ 1:** Админ-панель → кнопка "Удалить" → подтвердить

**Способ 2:** Страница новости → "🗑️ Удалить" → подтвердить

## 🔄 Обновление списка

На главной странице нажать:
```
🔄 Обновить список новостей
```

## 📊 API Reference

### Получить все новости
```javascript
GET /api/news
GET /api/news?category=current
```

### Получить одну новость
```javascript
GET /api/news/:id
```

### Создать новость
```javascript
POST /api/news
Body: {
  title: "Заголовок",
  content: "Текст новости",
  image: "URL",
  actor: "username"
}
```

### Обновить новость
```javascript
PUT /api/news/:id
Body: {
  title: "Новый заголовок",
  content: "Новый текст",
  actor: "username"
}
```

### Удалить новость
```javascript
DELETE /api/news/:id?actor=username
```

## 🎯 Горячие клавиши

- **F12** - Консоль разработчика
- **F5** - Обновить страницу
- **Ctrl+Shift+R** - Обновить с очисткой кэша
- **Ctrl+S** - Сохранить (в редакторе)

## ⚠️ Частые проблемы

| Проблема | Решение |
|----------|---------|
| Новости не грузятся | Проверить, запущен ли сервер |
| Кнопки не видны | Проверить is_admin в localStorage |
| Ошибка сохранения | Проверить консоль (F12) |
| Текст не разбивается | Использовать пустую строку между абзацами |

## 💡 Советы

1. **Всегда обновляйте список** после изменений в админ-панели
2. **Используйте прямые ссылки** для быстрого редактирования
3. **Проверяйте консоль** при ошибках
4. **Делайте резервные копии** news.json перед массовыми изменениями
5. **Тестируйте на тестовой новости** перед важными изменениями
