### Технические улучшения

1. **Обработка текста:**
   - Строки автоматически разбиваются на абзацы по двойному переносу строки
   - Массивы абзацев корректно объединяются для редактирования
   - Сохраняется форматирование при редактировании

2. **Обновление данных:**
   - Админ-панель автоматически обновляет список после изменений
   - Главная страница имеет кнопку ручного обновления
   - Дата новости обновляется при редактировании

3. **Пользовательский опыт:**
   - Прямые ссылки на редактирование конкретных новостей
   - Кнопки управления прямо на странице просмотра
   - Подтверждение перед удалением
   - Информативные сообщения об ошибках

4. **Безопасность:**
   - Проверка прав администратора на всех уровнях
   - Валидация actor при каждом запросе
   - Защита от несанкционированного доступа

### Известные особенности
- При создании новости через API content может быть строкой или массивом
- При редактировании дата автоматически обновляется на текущую
- Preview и lead генерируются автоматически если не указаны
- Для работы кнопок редактирования нужен флаг is_admin=1 в localStorage
