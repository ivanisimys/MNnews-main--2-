# Быстрый старт - Тестирование изменений

## 1. Запуск сервера

Откройте терминал и выполните:
```bash
cd "C:\Users\PC\Documents\MNnews-main (2)\MNnews-main\server"
npm start
```

Вы должны увидеть: `MNnews server started: http://localhost:3000`

## 2. Проверка работы API

Откройте браузер и перейдите по адресу:
```
http://localhost:3000/api/health
```

Должен вернуться JSON с датой.

## 3. Просмотр новостей

Откройте главную страницу:
```
http://localhost:3000/index.html
```

Проверьте:
- [ ] Новости загружаются и отображаются
- [ ] Внизу есть кнопка "🔄 Обновить список новостей"
- [ ] После нажатия на кнопку новости обновляются
- [ ] Внизу списка появляется время обновления

## 4. Тестирование админ-панели

### Вход как администратор

1. Откройте консоль браузера (F12)
2. Выполните:
```javascript
localStorage.setItem('name', 'admin');
localStorage.setItem('is_admin', '1');
```

Или войдите через форму входа с учетными данными админа.

### Проверка админ-панели

Перейдите:
```
http://localhost:3000/pages/menu_pages/admin_panel.html
```

Проверьте:
- [ ] Список новостей загружается
- [ ] Кнопки "Изменить" и "Удалить" работают
- [ ] Кнопка "+ Создать новую новость" открывает редактор

### Создание новости

1. Нажмите "+ Создать новую новость"
2. Заполните:
   - Заголовок: "Тестовая новость"
   - Изображение: `https://placehold.co/500x400?text=Test`
   - Текст: 
   ```
   Это первый абзац тестовой новости.
   
   Это второй абзац с дополнительной информацией.
   
   Это третий заключительный абзац.
   ```
3. Нажмите "Сохранить изменения"
4. Проверьте, что новость появилась в списке

### Редактирование новости

**Способ 1:** Через админ-панель
1. Нажмите "Изменить" рядом с любой новостью
2. Измените заголовок (добавьте "[ОБНОВЛЕНО]")
3. Нажмите "Сохранить изменения"
4. Проверьте обновление в списке

**Способ 2:** Со страницы новости
1. Откройте любую новость (кликните на неё)
2. Прокрутите вниз - должны быть кнопки администратора
3. Нажмите "✏️ Редактировать новость"
4. Вас перенаправит в админ-панель с открытым редактором
5. Внесите изменения и сохраните

### Удаление новости

1. В админ-панели нажмите "Удалить" рядом с тестовой новостью
2. Подтвердите удаление
3. Проверьте, что новость исчезла из списка

## 5. Проверка детального просмотра

1. Откройте главную страницу
2. Кликните на любую новость
3. На странице детали проверьте:
   - [ ] Отображается полное содержание новости
   - [ ] Если вы админ - видны кнопки редактирования/удаления
   - [ ] Кнопка "← Вернуться к списку новостей" работает

### Тест удаления со страницы детали

1. Будучи админом, откройте тестовую новость
2. Нажмите "🗑️ Удалить новость"
3. Подтвердите удаление
4. Должно произойти перенаправление на главную страницу

## 6. Проверка архивных страниц

```
http://localhost:3000/pages/news_archive1.html
http://localhost:3000/pages/news_archive2.html
```

Проверьте:
- [ ] Новости соответствующей категории загружаются
- [ ] Кнопка обновления работает (если добавлена)

## 7. Тестирование обработки ошибок

### Сервер выключен
1. Остановите сервер (Ctrl+C)
2. Обновите страницу
3. Должно появиться сообщение об ошибке с просьбой запустить сервер

### Неверные данные
1. Попробуйте создать новость без заголовка
2. Должна появиться ошибка валидации

## Возможные проблемы и решения

### Проблема: Новости не загружаются
**Решение:** Убедитесь, что сервер запущен на порту 3000

### Проблема: Кнопки админа не видны
**Решение:** Проверьте в консоли:
```javascript
console.log(localStorage.getItem('is_admin'));
// Должно быть "1"
```

### Проблема: Ошибка при сохранении
**Решение:** Откройте консоль браузера (F12) и посмотрите ошибку в Network tab

### Проблема: Content не разбивается на абзацы
**Решение:** Используйте двойной перенос строки (пустая строка между абзацами)
