ИНСТРУКЦИЯ ПО ПРОВЕРКЕ:
================================================================================

1. Проверка исправления дублирования уведомлений:
   ------------------------------------------------
   a) Запустите сервер: cd server && npm start
   b) Откройте http://localhost:3000
   c) Нажмите кнопку "🔄 Обновить список новостей" несколько раз
   d) Убедитесь, что надпись "Обновлено: [время]" только ОДНА
   e) Время должно обновляться при каждом нажатии

2. Проверка редактирования новостей:
   -----------------------------------
   a) Войдите как администратор
   b) Откройте любую новость
   c) Нажмите кнопку "✏️ Редактировать новость"
   d) Внесите изменения и сохраните
   e) Убедитесь, что изменения применились

3. Проверка регистрации (на Ubuntu):
   ----------------------------------
   a) Запустите setup_permissions.sh
   b) Зарегистрируйте нового пользователя
   c) Войдите как админ и проверьте заявки
   d) Примите заявку и убедитесь, что пользователь активен
