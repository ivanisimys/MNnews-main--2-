## Как тестировать:

### 1. Запуск сервера
```bash
cd "C:\Users\PC\Documents\MNnews-main (2)\MNnews-main\server"
npm start
```

### 2. Проверка категорий в браузере
1. Откройте `http://localhost:3000`
2. Войдите как администратор
3. Перейдите в админ-панель
4. Нажмите кнопку "Управление категориями"
5. Создайте новую категорию (например, "Тестовая категория")
6. Обновите страницу
7. В боковом меню должна появиться новая категория
8. Кликните на неё - должна открыться страница с новостями этой категории

### 3. Проверка существующих категорий
- "Главная страница" → `index.html`
- "Архив новостей 1" → `pages/category.html?id=archive1`
- "Архив новостей 2" → `pages/category.html?id=archive2`
- Новые категории → `pages/category.html?id={id_категории}`

### 4. Проверка создания новостей
1. Создайте новость и выберите новую категорию
2. Новость должна появиться на странице этой категории
3. Если на главной больше 6 новостей, старые должны автоматически переместиться в архив

### 5. Проверка удаления категорий
1. Удалите созданную тестовую категорию
2. Новости из этой категории должны переместиться в archive1 или current
3. Категория должна исчезнуть из бокового меню

## Возможные проблемы и решения:

### Проблема: Категории не открываются
**Решение:** 
- Проверьте консоль браузера (F12) на ошибки
- Убедитесь, что сервер запущен
- Проверьте, что файл `pages/category.html` существует

### Проблема: Меню не обновляется
**Решение:**
- Очистите кэш браузера (Ctrl+Shift+R)
- Проверьте, что файл `js/dynamic-menu.js` подключен на странице

### Проблема: Новости не загружаются на странице категории
**Решение:**
- Проверьте URL - должен быть параметр `?id={category_id}`
- Проверьте консоль на ошибки API запросов
- Убедитесь, что в `data/categories.json` есть эта категория

## API Endpoints для категорий:

- `GET /api/categories` - получить список категорий
- `POST /api/categories` - создать категорию
- `DELETE /api/categories/:id?actor={username}` - удалить категорию

Пример создания категории:
```json
{
  "name": "Новая категория",
  "description": "Описание",
  "actor": "admin"
}
```
