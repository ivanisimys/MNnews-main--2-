# Инструкция по новым функциям

## Что было добавлено:

### 1. Шапка сайта по центру
- Логотип и меню теперь центрированы на всех страницах

### 2. Автоматическое архивирование новостей
- При создании новой новости, если на главной странице больше 6 новостей, самая старая автоматически перемещается в архив
- По умолчанию старые новости перемещаются в "Архив новостей 1"
- Автоматическое архивирование происходит также при удалении новости

### 3. Управление категориями
**Создание категории:**
1. Войдите в админ-панель
2. Нажмите кнопку "📁 Управление категориями"
3. Введите название категории (обязательно)
4. Введите описание (необязательно)
5. Нажмите "+ Создать категорию"

**Удаление категории:**
1. Откройте "Управление категориями"
2. Нажмите кнопку "Удалить" напротив нужной категории
3. Подтвердите удаление
4. Все новости из удаляемой категории будут перемещены в "Архив новостей 1"

**Важно:**
- Системную категорию "Главная страница" (current) удалить нельзя
- При удалении категории все новости автоматически перемещаются в архив

### 4. Новости в архиве по 3 в ряд
- На страницах архива новости отображаются по 3 в ряд
- На планшетах (до 1024px) - по 2 в ряд
- На мобильных (до 700px) - по 1 в ряд

### 5. Динамическое меню
- Боковое меню теперь автоматически обновляется при создании/удалении категорий
- Не нужно вручную редактировать HTML для добавления новых разделов

## API endpoints для категорий:

```
GET /api/categories - Получить список всех категорий
POST /api/categories - Создать новую категорию
  Body: { name: "Название", description: "Описание", actor: "username" }
DELETE /api/categories/:id?actor=username - Удалить категорию
```

## Технические детали:

### Файлы данных:
- `data/categories.json` - хранит список категорий
- `data/news.json` - хранит новости с указанием категории

### Автоматическое архивирование:
- Максимум новостей на главной: 6
- Целевая категория для архива: archive1 (первый доступный архив)
- Срабатывает при создании и удалении новости

### CSS классы:
- `.archive-grid` - применяется к контейнеру новостей для отображения 3 в ряд
- `header { justify-content: center; }` - центрирует шапку

## Как протестировать:

1. Запустите сервер: `cd server && npm start`
2. Войдите как администратор
3. Создайте несколько новостей (больше 6)
4. Проверьте, что старые новости автоматически переместились в архив
5. Откройте "Управление категориями" и создайте новую категорию
6. Проверьте, что новая категория появилась в боковом меню
7. Откройте страницу архива - новости должны отображаться по 3 в ряд
