# Исправление дублирования уведомлений об обновлении

## Проблема

При нажатии на кнопку "🔄 Обновить список новостей" создавалось множество одинаковых уведомлений с временем обновления, вместо одного.

### Причина

Каждый раз при вызове функции `initNewsList()` создавался новый элемент `<p class="last-update">` с информацией о времени обновления, но старые элементы не удалялись. Это приводило к накоплению множества одинаковых надписей.

## Решение

Добавлена проверка и удаление старого элемента перед созданием нового.

### Измененный файл

**`js/news-list.js`** - функция `initNewsList()`

### Код исправления

```javascript
// Удаляем старое уведомление об обновлении, если оно есть
const existingUpdate = container.parentElement.querySelector('.last-update');
if (existingUpdate) {
    existingUpdate.remove();
}

// Добавляем информацию о последнем обновлении (только одну)
const lastUpdate = document.createElement('p');
lastUpdate.className = 'last-update';
lastUpdate.style.cssText = 'text-align: right; color: #888; font-size: 0.85em; margin-top: 10px;';
lastUpdate.textContent = `Обновлено: ${new Date().toLocaleTimeString('ru-RU')}`;
container.parentElement.appendChild(lastUpdate);
```

## Результат

✅ Теперь при каждом обновлении списка новостей отображается только **одно** уведомление с актуальным временем последнего обновления.

✅ Старые уведомления автоматически удаляются перед созданием новых.

## Тестирование

1. Откройте главную страницу (`index.html`)
2. Нажмите кнопку "🔄 Обновить список новостей"
3. Убедитесь, что появилась надпись "Обновлено: [время]"
4. Нажмите кнопку еще несколько раз
5. Убедитесь, что надпись всегда только одна и время обновляется

## Примечание

Это исправление влияет только на визуальное отображение времени обновления и не затрагивает функциональность загрузки новостей.
